# Linked List 项目说明

本项目是一个使用 JavaScript 实现的单向链表 (Singly Linked List) 数据结构。

## 功能特性

该 `Linkedlist` 工厂函数提供了以下用于操作链表的方法：

- `append(value)`: 在链表末尾添加一个包含 `value` 的新节点。
- `prepend(value)`: 在链表开头添加一个包含 `value` 的新节点。
- `size()`: 返回链表中节点的总数。
- `head()`: 返回链表第一个节点的值。
- `tail()`: 返回链表最后一个节点的值。
- `at(index)`: 返回指定索引（index）处的节点值。
- `pop()`: 移除链表的最后一个元素。
- `contains(value)`: 检查链表中是否包含某个值，如果包含则返回 `true`，否则返回 `false`。
- `find(value)`: 返回包含该值的节点的索引，如果未找到则返回 `false`。
- `toString()`: 以数组形式打印链表中的所有值。
- `insertAt(value, index)`: 在指定的索引位置插入一个包含 `value` 的新节点。
- `removeAt(index)`: 移除指定索引位置的节点。
